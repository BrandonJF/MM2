Search Help

To search the directory of active members, enter criteria into one field or a combination of the fields above, and then click Submit.  Different ways to search for John Smith include:   

1. If you know how to spell the name of the member you are looking for, type in the Last Name and First Name above.  This will search the directory and display exact matches.

2. If you are not sure how to spell a member's name, you can do a pattern match using the % wild card character. For example, type "Sm%" in the Last Name field and type "John" in the First Name field.  This will give you a list of members with the first name "John" and a last name starting with "Sm". Note: do not type in the quotes.
